## Features:
- Integration with CC
- Creates solution file
- Open file in editor
- Library injection
- Testing and profiling
- Sandboxed running 

## Usage
```bash
cocom [options]

options:
  --config-file <file>: Path to config file
```

### Shortcuts
```yaml
- r: run
- f: create file
- e: show error
- d: show diff 
- n: new test case
- a: answer + output
- i: input + answer (default)
- <space>: toggle output
```
